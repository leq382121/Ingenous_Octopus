# Ingenous_Octopus

Welcome to the rabbit hole. 

### to install the project, run this "sausage"

```
npm i && cd frontend\ && npm i && cd ../server && npm i && cd ..
```

### Then, simply run start command while in the root.

```
npm run start
```

Voila.
